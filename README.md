# Touch Spread Hedge Calculator

Static GitHub Pages version of the if-touch spread hedge calculator.

## Publish

1. Create a public GitHub repository named `touch-spread-calculator`.
2. Put these files at the repository root:
   - `index.html`
   - `README.md`
3. Push to `main`.
4. In GitHub, open `Settings` -> `Pages`.
5. Set:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Open:

```text
https://pohaoc2.github.io/touch-spread-calculator/
```

On iPhone Safari, open the URL, tap Share, then Add to Home Screen.
